# SMW Website Clone

Современный одностраничный сайт, созданный на основе дизайна SMW. Включает все основные секции и функциональность оригинального сайта.

## Особенности

- **Адаптивный дизайн** - полностью адаптирован для мобильных устройств, планшетов и десктопов
- **Интерактивная карусель** - главный слайдер с автоматической прокруткой и управлением
- **Аккордеон секция** - для отображения информации о жизненном цикле продукта
- **Табы** - для переключения между различными возможностями компании
- **Плавная прокрутка** - навигация по якорным ссылкам с плавной прокруткой
- **Мобильное меню** - адаптивное меню для мобильных устройств
- **SEO оптимизация** - семантическая разметка и доступность

## Структура проекта

```
.
├── index.html          # Главная HTML страница
├── styles.css          # Все стили сайта
├── script.js           # JavaScript функциональность
├── assets/             # Папка для изображений
│   ├── logo.png        # Логотип компании
│   ├── placeholder-*.jpg # Изображения для секций
│   └── munich.jpg      # Изображение офиса
└── README.md           # Документация

```

## Установка и запуск

1. Клонируйте или скачайте проект
2. Откройте `index.html` в браузере
3. Или используйте локальный сервер:

```bash
# С Python
python -m http.server 8000

# С Node.js (http-server)
npx http-server

# С PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

## Секции сайта

1. **Header** - Навигация и языковой переключатель
2. **Hero Carousel** - Главный слайдер с призывом к действию
3. **Marketsegments** - Карточки с рыночными сегментами
4. **Product Lifecycle** - Аккордеон с этапами жизненного цикла
5. **Capabilities** - Табы с возможностями компании
6. **Wheels** - Информация о колесах
7. **Company** - О компании и контактная информация
8. **Footer** - Социальные сети и ссылки

## Кастомизация

### Изменение цветов

Отредактируйте CSS переменные в `styles.css`:

```css
:root {
    --primary-color: #333;
    --secondary-color: #666;
    --accent-color: #000;
    /* и т.д. */
}
```

### Добавление изображений

Поместите изображения в папку `assets/` и обновите пути в `index.html`.

### Изменение контента

Отредактируйте текст напрямую в `index.html`.
