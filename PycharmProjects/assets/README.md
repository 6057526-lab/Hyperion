## Временные заглушки

Если у вас нет изображений, вы можете использовать placeholder сервисы:
- https://via.placeholder.com/
- https://placehold.co/
- https://picsum.photos/

Пример:
- `https://via.placeholder.com/800x600?text=Automotive` для placeholder-1.jpg
- `https://via.placeholder.com/200x50?text=LOGO` для logo.png
